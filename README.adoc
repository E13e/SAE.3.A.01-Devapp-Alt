== Introduction

toc::[]

Dans ce projet, les objectifs sont :

	* D'afficher les informations des capteurs des 18 panneaux solaires de l'IUT de Blagnac (au-dessus de la bibliothèque) dans des graphiques sur une page web.

	* De mettre en place un système d'alerte de surconsommation d'électricité dans une salle, avec un principe d'acquittement de l'alerte.
